# 맡겨멍 서비스 기획서

## 1. 프로젝트 개요

| 항목 | 내용 |
| --- | --- |
| 프로젝트명 | 맡겨멍 |
| 프로젝트 소개 | 맡겨멍은 반려견을 맡길 수 있는 애견 유치원 및 호텔 정보를 지도 기반으로 탐색하고 예약할 수 있는 플랫폼 서비스이다. |

보호자는 주변 업체를 검색하고 예약할 수 있으며, 강아지를 맡긴 이후에도 업체가 작성하는 알림장과 실시간 채팅을 통해 반려견의 상태를 확인할 수 있다.

단순 예약 서비스를 넘어 반려견 케어 기록과 소통 기능을 통합 제공하여 보호자의 불안감을 줄이는 것을 목표로 한다.

## 2. 프로젝트 필요성

현재 보호자들은 다음과 같은 문제를 겪고 있다.

- 여러 지도 앱을 통해 업체를 직접 검색해야 함
- 예약 과정이 전화 및 카카오톡 중심이라 불편함
- 맡긴 이후 강아지 상태를 지속적으로 확인하기 어려움
- 사진과 기록이 체계적으로 저장되지 않음
- 업체 정보와 실제 리뷰를 한곳에서 비교하기 어려움

맡겨멍은 이러한 문제를 해결하기 위해 다음 기능들을 제공한다.

- 지도 기반 업체 탐색
- 예약 시스템
- 실시간 알림장
- 보호자 ↔ 업체 실시간 채팅
- 실제 이용자 리뷰
- 푸시 알림 기반 실시간 알림

## 3. 핵심 타겟

### 3-1. 1인 가구 보호자

출장 및 외출이 잦아 반려견을 잠시 맡겨야 하는 사용자

### 3-2. 장기 여행 사용자

여행 및 장기 일정으로 인해 며칠 이상 반려견을 맡길 장소가 필요한 사용자

### 3-3. 정기 유치원 이용자

정기적으로 반려견 유치원을 이용하려는 사용자

예시:

- 평일 오후 2시 ~ 5시 정기 이용

## 4. 핵심 기능

### 4-1. 지도 기반 업체 검색

#### 기능 설명

현재 위치를 기반으로 주변 애견호텔 및 애견유치원을 지도에 표시한다.

#### 제공 기능

- 현재 위치 표시
- 주변 업체 마커 표시
- 지도 이동 시 재검색
- 업체 상세 페이지 이동

#### 필터 기능

- 대형견 가능
- 픽업 가능

### 4-2. 업체 상세 페이지

#### 제공 정보

- 업체 사진
- 소개글
- 가격
- 운영시간
- 리뷰
- 시설 정보
- 픽업 서비스 여부
- 운동장 유무

### 4-3. 예약 기능

#### 예약 방식

업체 승인 후 예약 확정 방식

#### 지원 예약 유형

일반 예약

- 특정 기간 맡김

예시:

- 5월 10일 ~ 5월 12일

정기권 예약

- 반복 예약 기능

예시:

- 평일 오후 2시 ~ 5시

#### 예약 흐름

1. 날짜 선택
2. 예약 요청
3. 업체 승인
4. 예약 확정

### 4-4. 알림장 기능

#### 기능 설명

업체가 보호자에게 강아지 상태를 기록하여 전달한다.

#### 작성 가능 형태

- 사진
- 영상
- 텍스트
- 사진 + 텍스트

#### 특징

- 보호자만 열람 가능
- 맡김 종료 후에도 기록 유지
- 날짜별 조회 가능
- 실시간 앱 푸시 알림 제공

### 4-5. 실시간 채팅

#### 기능 설명

보호자와 업체 간 1:1 채팅 제공

#### 제공 기능

- 텍스트 채팅
- 예약 관련 자동 메시지
- 앱 푸시 알림 제공

#### 자동 메시지 예시

- 예약 요청 완료
- 예약 승인 완료
- 예약 거절 안내
- 체크인 완료
- 체크아웃 완료

### 4-6. 리뷰 시스템

#### 작성 조건

실제 예약 완료 사용자만 작성 가능

#### 리뷰 요소

- 별점
- 사진 첨부
- 재방문 여부
- 후기 작성

## 5. 차별성

### 5-1. 맡긴 이후까지 관리

기존 서비스는 예약 중심 기능만 제공하는 경우가 많다.

맡겨멍은:

- 실시간 알림장
- 사진 및 영상 기록
- 채팅 기능
- 푸시 알림

을 통해 보호자의 불안감을 줄인다.

### 5-2. 체계적인 반려 기록 관리

기존에는 카카오톡으로 전달받은 내용이 시간이 지나면 사라지는 문제가 있었다.

맡겨멍은:

- 알림장
- 사진
- 영상
- 예약 기록
- 채팅 기록

등을 앱 내부에 저장하여 지속적으로 관리 가능하다.

### 5-3. 반려견 위탁 특화 플랫폼

일반 지도 서비스와 달리:

- 대형견 가능 여부
- 픽업 가능 여부
- 운동장 유무

등 반려견 위탁에 필요한 정보를 중심으로 제공한다.

맡겨멍은 단순한 애견호텔 예약 서비스를 넘어 보호자와 업체 간의 신뢰와 소통을 강화하는 반려견 위탁 통합 플랫폼이다.
